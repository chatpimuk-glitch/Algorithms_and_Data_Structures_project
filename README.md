วิธีใช้งาน
  โปรแกรมจะรอรับคำสั่งจากผู้ใช้
    เพิ่มรายวิชา ➤ add <CourseCode>
    ตัวอย่าง: add 10123102
    ลบรายวิชาล่าสุด ➤ undo
    ประมวลผลทั้งหมด ➤ process_all

การวิเคราะห์ Big-O
  การสร้าง Hash Table : O(n)
  การค้นหาข้อมูล (add) : O(1)
  ลบรายวิชาล่าสุด (undo) : O(1)
  ประมวลผลทั้งหมด (process_all) : O(nlog(n))
